# Amazon Transfer Family (amazon-transfer-family)

AWS Transfer Family is a secure transfer service that enables you to transfer files into and out of AWS storage services. It supports SFTP, FTPS, and FTP protocols, providing a fully managed file transfer service with native integration to Amazon S3 and Amazon EFS.

## APIs

### AWS Transfer Family API

The AWS Transfer Family API provides programmatic access to create and manage SFTP, FTPS, and FTP file transfer servers. It enables managing users, SSH public keys, roles, and server configurations for secure file transfer workflows integrated with Amazon S3 and EFS.
